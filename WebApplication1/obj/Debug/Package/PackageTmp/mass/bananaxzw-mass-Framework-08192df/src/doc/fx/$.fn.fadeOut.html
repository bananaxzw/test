$.fn.fadeOut(duration, props)

描述：

让匹配的每个节点进行淡出动画。

参数：

duration
可选。Number。动画执行时间，单位为ms，默认为500ms。
可选。Function。动画结束时的回调函数
props
可选。Object。各种配置。

返回值：

mass实例

props配置对象可以指定各种CSS属性，如width, height, color, fontSize, opacity等进行动画变化， 还支持scrollTop与scrollLeft这两个属性进行动画变化。还可以指定动画开始前的回调函数before,动画结束时的回调函数after, 动画是否进行倒退reverse....

例子 
                $.require("ready,fx,event",function(){
                    $("#btn1").click(function() {
                        function complete(el) {
                            $("<div/>").text(el.id).appendTo("#log");
                        }
                        $("#box1").fadeOut(1600, { o:[0,"linear"],after:[ complete ]});
                        $("#box2").fadeOut(1600, { after:complete});
                    });

                    $("#btn2").click(function() {
                        $("#box1,#box2").show();
                        $("#log").empty();
                    });
                });
