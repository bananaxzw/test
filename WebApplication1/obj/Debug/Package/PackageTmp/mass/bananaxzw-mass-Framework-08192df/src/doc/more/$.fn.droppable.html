$.fn.droppable(settings)

描述：

拖放组件的第二个部件，为拖动对象增加“可投掷”功能，用于处理拖动块与靶场的关系。

参数：

settings
可选。用于配置控件对象。但当这个对象第二次调用它时， 只有当其第一个参数为字符串或对象才有效，这样相当于调用其方法或重写一些属性。

返回值：

mass实例

settings的配置参数：

  • range：用于圈定靶场的位置
    • 如果是一个CSS表达式，将转化一个mass的节点链对象。
    • 如果是一个mass的节点链对象,不处理。
    • 如果是一个元素节点，将转化一个mass的节点链对象。
    • 如果是一个元素数组，将转化一个mass的节点链对象
    建议在dragstart时设置。
  • hoverClass： 当光标进入靶场时，为靶场添加的类名。
  • activeClass：当拖动块移动时，为靶场添加的类名,提示用户要把它投掷到这里。
  • dragenter：function 当光标进入靶场时，执行此回调。
  • dragover：function 当光标进入靶场后，执行此回调。
  • dragleave：function 当光标进入靶场时，执行此回调。
  • drop：function 当光标进入靶场后，留在其上移动时放开鼠标时，执行此回调。
  • tolerance：function/string 自定义触发模式,默认是pointer模式，也即W3C的模式,当鼠标进入靶场才触发dragenter回调,离开则触发dragleave回调。 此外还有如下模式可供选择:
    • intersect: 当拖动块与靶场存在重叠部分，就触发dragenter回调。
    • middle: 只有当拖动块有一半的面积进入靶场，才触发dragenter回调。
    • fit: 只有当拖动块完全位于靶场之内，才触发dragenter回调。

dragenter, dragover, dragleave, drop这三个都是HTML5标准的原生事件名称。

生成拖动对象比原生多了以下属性：

  • droppers：一个对象数组，里面装载的是靶场的坐标宽高的消息（elem, width, height, top, bottom, left, right）
  • dropper：mass实例，当前光标进入的靶场
