位置伪类，:first, :last, :even, :odd, :eq(n), :gt(n), :lt(n)。

它是根据上一个表达式得到的节点集合的索引值来选择元素的。如"div:first",它会得到一组DIV元素，然后取得第一个返回。

此种伪类是非标准的，因此不建议使用。使用mass实例的同名方法，效果一致的。

例子
请点击下面表格
$("li:first").css("background-color","#FF9");
$("li:last").css("background-color","#6495ED");
$("li:even").css("background-color","#FF1493");
$("li:odd").css("background-color","#800080");
$("li:lt(4)").css("background-color","#E9967A");
$("li:gt(5)").css("background-color","#98FB98");
$("li:eq(3)").css("background-color","#FFF0F5");
