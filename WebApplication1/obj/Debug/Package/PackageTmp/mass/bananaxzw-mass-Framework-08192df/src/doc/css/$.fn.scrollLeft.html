$.fn.scrollLeft( [val])

描述：

取得或设置匹配节点的scrollLeft。

此方法是属性于get one set all操作，根据传入参数决定其是读操作还是写操作。

如果没有参数，则读取第一个元素的scrollLeft，并返回。

如果存在参数，通过它设置所有元素的scrollLeft,并返回原实例。

例子

lalala

Hello

                <style>
                div.demo {
                     background:#CCCCCC none repeat scroll 0 0;
                     border:3px solid #666666;
                     margin:5px;
                     padding:5px;
                     position:relative;
                     width:200px;
                     height:100px;
                     overflow:auto;
                }
                .demo p { margin:10px;padding:5px;border:2px solid #666;width:1000px;height:1000px; }
                </style>

                <div class="demo"><h1>lalala</h1><p>Hello</p></div>

                
            $.require("ready,node,css,event",function(){
                $("div.demo").scrollLeft(300)
            })
