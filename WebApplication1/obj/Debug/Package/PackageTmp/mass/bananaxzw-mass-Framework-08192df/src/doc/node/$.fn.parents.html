$.fn.parents([expr])

描述：

把匹配元素的所有祖先收集起来，去重排序组成新mass实例返回。

参数：

expr
可选，String，用于进一步过滤的CSS选择器。

返回值：

新mass实例

例子

显示.sample1的所有孩子的层次关系

显示其所有祖先: 

                <div class="sample1" style="background:#ffe0bb;color:#000;">
                    <div>
                        <p>
                            <span>
                                <kbd>显示其所有祖先: </kbd>
                            </span>
                        </p>
                    </div>
                </div>
                
                $.require("ready,node",function(){
                    var self = $("kbd")
                    var tags = self.parents().collect(function () { 
                        return this.tagName.toLowerCase(); 
                    }).join(", ");
                    self.append("" + tags + "");
                });
