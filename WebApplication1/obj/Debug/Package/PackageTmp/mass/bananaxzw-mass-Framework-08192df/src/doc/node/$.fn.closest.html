$.fn.closest(expr, [context])

描述：

取得匹配节点中所有第一个匹配expr的最近祖先，去重排序后构成新实例返回。

参数：

expr
必需。String。CSS表达式。
context
可选。元素节点。

返回值：

新mass实例。

例子

closest最大的用处是实现自己的事件代理，虽然事件代理在dom Framework中提供支持。

  • list item 1
  • list item 2
    • list item 2-a
    • list item 2-b
  • list item 3
  • list item 4
                <ul class="sample1">
                    <li>list item 1</li>
                    <li>list item 2
                        <ul>
                            <li>list item 2-a</li>
                            <li>list item 2-b</li>
                        </ul>
                    </li>
                    <li><b>list item 3</b></li>
                    <li>list item 4</li>
                </ul>
                
            $.require("ready,attr,event",function(){
                $(document).bind("click", function (e) {
                    $(e.target).closest("li").toggleClass("hilight");
                });
            });
