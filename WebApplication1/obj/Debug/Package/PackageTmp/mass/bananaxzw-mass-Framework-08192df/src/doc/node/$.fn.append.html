$.fn.append(content)

描述：

向每个匹配的元素内部追加内容。

这个操作与对指定的元素执行appendChild方法，将它们添加到文档中的情况类似。

参数：

content
必需，可以为元素节点，符合套嵌关系的HTML片断，CSS选择器或dom对象。

返回值：

原mass实例
