object.without(arr)

描述：

返回当前对象的一个子集，它不包含arr所指定的键名。

arr
必需。要去掉的键名的集合。
例子 
                $.require("lang",function(){
                    var obj = {
                        a:"one",
                        b:"two",
                        c:"three"
                    };
                    var neo = $.lang(obj).without(["a","b"]);
                    $.log($.dump(neo), true);
                });
