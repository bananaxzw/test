状态伪类，只能针对表单元素。下面几个只有selected是自定义的，其他都是CSS3的标准伪类。

  • :enabled匹配所有可用元素。
  • :disabled匹配所有不可用元素。
  • :checked匹配所有选中的被选中元素(复选框、单选框等，不包括select中的option)
  • :selected 匹配所有选中的option元素
  • :indeterminate 匹配所有处于不确定状态的checkbox元素（HTML5属性 或FF官网）
例子
请点击下面表格
$("input:enabled").css("background-color","#F542D7");
$("input:disabled").css("background-color","#6495ED");
alert($("input:checked").collect(function(){ return this.value }).join(" ")); //Daily Monthly
alert($("select option:selected").val())
$(":indeterminate+span").css("background-color","limegreen");
