$.fn.click([ fn ])

描述：

为 click 事件绑定一个处理函数，或者触发元素上的 click 事件。

这个函数的第一种用法是 .bind('click', handler) 的快捷方式，第二种用法是 .fire('click') 的快捷方式。

参数：

fn
可选。Funcion。回调函数。
例子

点击第一个按钮只会为自己加1，点击第二个按钮会为两者加1

0 button #1 clicks.
0 button #2 clicks.
            <div class="sample1">
                <button>Button #1</button>
                <button>Button #2</button>
                <div><span>0</span> button #1 clicks.</div>
                <div><span>0</span> button #2 clicks.</div>
            </div>
            
        $.require("ready,event",function(){
            var spans = $(".sample1 span")
            var firstBtn = $(".sample1 button").first().click(function () {
                update(spans.first());
            });
            $(".sample1 button").last().click(function () {
                firstBtn.fire('click');
                update(spans.last());
            });

            function update(el) {
                var n = parseInt(el.text(), 10);
                el.text(n + 1);
            }
        });
