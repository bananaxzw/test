元素包含伪类，:has(expr)。如果元素含有某个特殊的孩子（此孩子符合has给出的表达式），那么此元素被选中。

:has为非标准伪类，不推荐使用，建议使用has方法，能达到相同的效果。

例子
请点击下面表格
$(".sample div:has('strong')").css("background-color", "red"); 
                <div class="sample">
                    <div><strong>此DIV会变红</strong></div>

                    <div>这个不会</div>
                </div>
此DIV会变红
这个不会
