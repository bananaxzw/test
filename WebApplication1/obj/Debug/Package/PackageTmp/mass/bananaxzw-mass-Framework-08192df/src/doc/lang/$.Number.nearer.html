number.nearer(n1,n2)

描述：

求出距离原数最近的那个数

number
必需。Number。最小值或最大值。
fn
必需。Number。最大值或最小值。
例子 
                $.require("lang",function(){
                    var a = $.lang(6);
                    $.log(a.nearer(1,7),true);
                    $.log(a.nearer(-1,4),true);
                    $.log(a.nearer(3,4),true);
                });
