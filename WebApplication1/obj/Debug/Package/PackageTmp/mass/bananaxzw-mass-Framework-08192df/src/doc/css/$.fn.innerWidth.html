$.fn.innerWidth( )

描述：

取得第一个元素的当前宽度,包括padding，但是不包括border。

此方法是个只读文法。

此方法不适用于window 与 document对象，可以使用.width()代替。

w3c的盒子模型

例子 
 <style>   
      .innerWidth { margin:10px;padding:5px;border:2px solid #666; }
</style>
<dvi id="innerWidth"> </div>
