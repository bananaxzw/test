$.fn.replaceClass( old, neo)

描述：

如果匹配元素存在old类名则将它替换为neo类名，不存在则不会做任何操作的。

参数：

old
必须。String。要替换掉的类名。
neo
必须。String。用来替换的新类名。
例子
请点击它
                <div class="sample">请点击它</div>
                <style>
                    .selected {
                       background:#8B0000;
                    }
                </style>
                
            $.require("ready,event,attr",function(){
                $(".sample").click(function(){
                     $(this).replaceClass("sample","selected");
                });
            });
