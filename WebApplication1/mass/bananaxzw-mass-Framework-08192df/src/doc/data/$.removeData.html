$.removeData(obj,[ name ])

描述：

如果指定了name，则移除这个数据， 否则清空所有与之相关的数据。

参数：

obj
必需。不为属性节点或文本节点的其他类型节点，文档对象，window对象，自定义对象等等。
name
可选。String。数据的名字。

返回值：

如果指定了name，则会返回与之对应的值，否则为undefined。

例子 
                $.require("ready,data",function(){
                    var a = {};
                    $.data(a,"name","司徒正美");
                    $.log($.dump(a),true);
                    $.log($.removeData(a,"name"),true);
                    $.log($.dump(a),true);
                });
