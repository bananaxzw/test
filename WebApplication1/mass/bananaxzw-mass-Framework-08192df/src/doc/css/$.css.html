$.css(node, name, [ val ])

描述：

$.fn.css对应的静态方法，$.fn.css内部其实就是调用它。

参数：

nodes
必需。单个节点。
name
必需。第一种情况，String。数据的名字。如果只有它一个参数，则进行读操作，当前样式。
必需。第一种情况，一个普通的Object，进行写操作，同时对所有匹配元素设置多个样式，返回实例本身。
val
可选。样式的值。如果是数值，对于表示尺寸的样式，如width,height, fontSize等，会补上px单位，然后进行样式设置，返回实例本身

name为字符串时，一些样式名称可以使用缩写来代替，如"backgroundColor"可略写为"bgc"，下面是其一览表：

    var shortcuts = {
        c:   "color",
        h:   "height",

        r:   "rotate",
        w:   "width",
        x:   "left",

        sx:  "scaleX",
        sy:  "scaleY",
        tx:  "translateX",
        ty:  "translateY",
        bgc: "backgroundColor"
    };
例子 
                <div class="skimover">

                </div>

            
$.require("ready,node,css,event",function(){
     $(".skimover").hover(function(){
        $.css(this,"backgroundColor","#008080")
     },function(){
        $.css(this,"backgroundColor","transparent")
    });
});
