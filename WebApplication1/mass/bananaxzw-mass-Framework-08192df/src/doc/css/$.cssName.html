$.cssName(name)

描述：

用于取得正确可用的样式名称，在现代浏览器中，它们私自下实现了许多CSS3属性，都未必都能成为标准，因此都带着-moz-，-webkit-，-o-这样的前缀。本方法就是专门用于取得这样方法名。

参数：

name
必需。String。必须是连字符风格

返回值：

一个驼峰风格的样式名或null

例子 
//-ms-interpolation-mode: bicubic （可以让IE7的图片在有缩放的情况下，失真变得不怎样明显）
$.require("ready,node,css",function(){
    $.log($.cssName("interpolation-mode"),true)
});
