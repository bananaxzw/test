$.fn.outerWidth([includeMargin] )

描述：

取得第一个元素的当前宽度,包括padding，border和选择性的margin

此方法是个只读文法。

此方法不适用于window 与 document对象，可以使用.width()代替。

参数：

includeMargin
可选。Boolean。是否加上margin。

w3c的盒子模型

例子 
 <style>   
      .outerWidth { margin:10px;padding:5px;border:2px solid #666; }
</style>
