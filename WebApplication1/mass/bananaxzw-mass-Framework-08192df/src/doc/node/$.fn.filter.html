$.fn.filter( expr )

描述：

取得当前匹配节点的所有匹配expr的节点，组成新mass实例返回。

参数：

expr
必需。String|Function。CSS表达式，或者是一个回调函数。

返回值：

新的mass实例

例子

取得.sample1所有索引值为偶数的LI元素，这经常用于商场或新闻中隔行变色效果。

  • list item 1
  • list item 2
  • list item 3
  • list item 4
  • list item 5
  • list item 6
                <div class="sample1">
                    <ul>
                        <li>list item 1</li>
                        <li>list item 2</li>
                        <li>list item 3</li>
                        <li>list item 4</li>
                        <li>list item 5</li>
                        <li>list item 6</li>
                    </ul>
                </div>
                
                $.require("ready,css",function(){
                    $(".sample1 li").filter(":nth-child(even)").css("color","red")
                });
例子

使用过滤函数取得只包含一个strong元素的LI元素。

  • list item 1 - one strong tag
  • list item 2 - two strong tags
  • list item 3
  • list item 4
  • list item 5
  • list item 6
                <div class="sample2">
                    <ul>
                        <li><strong>list</strong> item 1 - one strong tag</li>
                        <li><strong>list</strong> item <strong>2</strong> - two <span>strong tags</span></li>
                        <li>list item 3</li>
                        <li>list item 4</li>
                        <li>list item 5</li>
                        <li>list item 6</li>
                    </ul>
                </div>
                
                $.require("ready,css",function(){
                    $(".sample2 li").filter(function(el,i){
                         return $("strong", el).length == 1
                    }).css("color","red")
                });

使用过滤函数，this是当时正在操作的DOM元素。参数有两个,el与this一样，i为el在mass实例的索引值。

例子

为中间四个加上红框。

                <style>
                    .sample3 div { width:60px; height:60px; margin:5px; float:left;  border:2px white solid;}
                </style>
                <div class="sample3">
                    <div></div>
                    <div class="middle"></div>
                    <div class="middle"></div>
                    <div class="middle"></div>
                    <div class="middle"></div>
                    <div></div>
                </div>
                
                $.require("ready,css",function(){
                    $(".sample3 div")
                    .css("background", "#c8ebcc")
                    .filter(".middle")
                    .css("border-color", "red");
                });
