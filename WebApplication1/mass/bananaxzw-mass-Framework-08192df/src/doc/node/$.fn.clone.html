$.fn.clone(dataAndEvents, deepDataAndEvents)

描述：

复制匹配的每一个节点并组成新mass实例返回。

参数：

dataAndEvents
可选。是否复则原节点的数据与事件到新节点中，默认false。
end
deepDataAndEvents。是否对其所有孩子都做这种复制数据与事件的操作，如果没有指定，则与dataAndEvents的值保持一致。
例子
  • 列表1
  • 列表2
  • 列表3
  • 列表4
      <ul class="container">
        <li class="inner" >列表1</li>
        <li class="inner" >列表2</li>
        <li class="inner" >列表3</li>
        <li class="inner" >列表4</li>
      </ul>
                
                $.require("ready,node",function(){
                    $(".container li").clone().appendTo(".container")
                    
                });
