$.fn.parent([expr])

描述：

把匹配元素的父元素收集起来，去重排序组成新mass实例返回。

参数：

expr
可选，String，用于进一步过滤的CSS选择器。

返回值：

新mass实例

例子

显示.sample1的所有孩子的层次关系

div, span, b

p, span, em

div, strong, span, em, b, b 
                 <div class="sample1" style="background:#ffe0bb;color:#000;">
                    <div>div, 
                        <span>span, </span>
                        <b>b </b>

                    </div>
                    <p>p, 
                        <span>span, 
                            <em>em </em>
                        </span>
                    </p>

                    <div>div, 
                        <strong>strong, 
                            <span>span, </span>
                            <em>em, 
                                <b>b, </b>
                            </em>

                        </strong>
                        <b>b </b>
                    </div>
                </div>
                
                $.require("ready,node",function(){
                    $(".sample1 *").each(function () {
                        var parentTag = $(this).parent()[0].tagName.toLowerCase();
                        $(this).prepend(document.createTextNode(parentTag + " > "));
                    });
                });
