$.fn.prev([expr])

描述：

把匹配元素的与之毗邻的上一个元素节点收集起来，去重排序组成新mass实例返回。

参数：

expr
可选，String，用于进一步过滤的CSS选择器。

返回值：

新mass实例

例子

将.third-item的上一个LI元素的背景变红。

  • list item 0
  • list item 1
  • list item 2
  • list item 3
  • list item 4
  • list item 5
  • list item 6
                <ul>
                    <li>list item 1</li>
                    <li>list item 1</li>
                    <li>list item 2</li>
                    <li class="third-item">list item 3</li>
                    <li>list item 4</li>
                    <li>list item 5</li>
                    <li>list item 6</li>
                </ul>
                
                $.require("ready,node",function(){
                    $(".third-item").prev().css("background-color","red");
                });
