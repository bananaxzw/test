$.fn.has( expr )

描述：

取得匹配节点中那些后代中能匹配给定CSS表达式的节点，组成新mass实例返回。

参数：

expr
必需。String。CSS表达式。

返回值：

新的mass实例

例子

让拥有子菜单的菜单字体变红。

  • list item 1
  • list item 2
    • list item 2-a
    • list item 2-b
  • list item 3
  • list item 4
                <ul class="sample1">
                    <li>list item 1</li>
                    <li>list item 2
                        <ul>
                            <li>list item 2-a</li>
                            <li>list item 2-b</li>
                        </ul>
                    </li>
                    <li><b>list item 3</b></li>
                    <li>list item 4</li>
                </ul>
                
                $.require("ready,css",function(){
                   $('.sample1 li').has('ul').css('color', 'red');
                });
