array.last([fn, score])

描述：

如果第一个参数为函数，则返回数组中最后一个符合此回调的元素，没有参数则返回最后一个元素

参数：

fn
可选。回调函数。
score
可选。要绑定的作用域。
例子 
                $.require("lang",function(){
                    var a = $.lang([1,2,3,4]);
                    $.log(a.last(),true);
                    $.log(a.last(function(el){return el < 4}),true)
                });
