$.dump

描述：

此方法可以将javascript数据用很漂亮的缩进打印出来。

传参：

一个任意类型的javascript对象。

返回值：

String。

例子 
                $.require("lang,support",function(){
                    var a = $.dump("XXXX" );
                    $.log(a,true);
                    var b = $.dump( $.support );
                    $.log(b,true);
                });
