array.remove(el)

描述：

破坏性方法，修改原对象。移除第一个匹配的元素。

参数：

el
必须。要移除的元素。

返回值：

被移除的元素或null。

例子 
                $.require("lang",function(){
                    var test = [3, 5, 6, 1, 20];
                    var a = $.lang(test);
                    $.log(a.remove(5),true);
                    $.log($.dump(test),true);
                });
