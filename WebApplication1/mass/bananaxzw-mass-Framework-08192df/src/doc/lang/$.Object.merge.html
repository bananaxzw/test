object.merge(obj1,[obj2],[objN])

描述：

将所有参数（必须为对象）全部赋给当前对象。

obj1
必需。要合并的对象1。
obj2
可选。要合并的对象2。
objN
可选。要合并的对象N。
例子 
                $.require("lang",function(){
                    var obj1 = {
                        a: 0,
                        b: 1
                    };
                    var obj2 = {
                        c: 2,
                        d: 3
                    };
                    var obj3 = {
                        a: 4,
                        d: 5
                    };
                    var merged  = $.lang(obj1).merge(obj2, obj3);
                    $.log($.dump(merged),true);
                });
