$.fn.keyup([ fn ])

描述：

为 keyup 事件绑定一个处理函数，或者触发元素上的 keyup 事件。

这个函数的第一种用法是 .bind('keyup', handler) 的快捷方式，第二种用法是 .fire('keyup') 的快捷方式。

javascript事件主要通过以下三个事件来捕获键盘事件：onkeydown,onkeypress与onkeyup。 该三个事件的执行顺序如下：onkeydown -> onkeypress ->onkeyup。

参数：

fn
可选。Funcion。回调函数。
例子

测试onkeydown,onkeypress与onkeyup的执行顺序.

            $.require("ready,event",function(){
                var callback = function(e){
                    $(".sample").height(  $(".sample div").append(e.type+"<br/&gr;").height()+30)
                }
                $(".sample input").bind({
                    keydown:callback,
                    keypress:callback,
                    keyup:callback
                })
            });
