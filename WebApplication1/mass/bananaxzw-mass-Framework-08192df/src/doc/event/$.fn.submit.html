$.fn.submit( fn)

描述：

为 submit 事件绑定一个处理函数，或者触发元素上的submit 事件。

这个函数的第一种用法是 .bind('submit', handler) 的快捷方式，第二种用法是 .fire('submit') 的快捷方式。

参数：

fn
必需。Funcion。回调函数。
例子
点击这里 
                <form id="target" action="$.fn.on.html">
                    <input type="text" value="随便填点什么" />
                    <input type="submit" value="提交" />
                </form>
                <div id="other"> 
                    点击这里
                </div>
                
            $.require("ready,event",function(){
                $('#target').submit(function() {
                    $(this).prepend('触发submit事件<br/>');
                    return false;
                });
                $('#other').click(function() {
                    $('#target').submit();
                });
            });
