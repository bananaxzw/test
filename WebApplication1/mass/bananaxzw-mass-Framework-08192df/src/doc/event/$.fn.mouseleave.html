$.fn.mouseleave([ fn ])

描述：

为 mouseenter（鼠标离开） 事件绑定一个处理函数，或者触发元素上的 mouseenter（鼠标离开） 事件。

这个函数的第一种用法是 .bind('mouseleave', handler) 的快捷方式，第二种用法是 .fire('mouseleave') 的快捷方式。

它与mouseout的区别是不会冒泡 。

参数：

fn
可选。Funcion。回调函数。
例子 
            $.require("ready,event",function(){
                $("#red").bind("mouseleave",function(e){
                    $.log("mouseleave回调1")
                });
                $("#red").bind("mouseleave",function(e){
                    $.log("mouseleave回调2")
                });
                $("#red").bind("mouseleave",function(e){
                    $.log("mouseleave回调3")
                });

                $("#red").fire("mouseleave")
            })
