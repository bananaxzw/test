$.fn.live( types, fn, [times])

描述：

通过监听器绑定到文档对象上，然后检测传播上来的事件源对象是否与现在匹配的节点的选择器相符，再决定是否触发事件。

每一个mass实例都有一个selector属性，我们就是能过它检测事件源对象。

此方法不但能对当前孩子有效，还能监听以后加入的新节点。

可点击这里，查看《bind，live，delegate方法的区别》。

参数：

types
必需。String。一个包含一个或多个JavaScript事件类型的字符串，比如"click"或"keydown,"或自定义事件的名称。没有则清空此元素所绑定的所有回调。
fn
必需。Funcion。回调函数。
times
可选。正整数。限制此回调执行多少次就自动卸载。
例子
Click!
            $.require("ready,event,fx",function(){
                function callback() {
                    $(".hide_div").show(1000);
                }
                $("#bind").click(function () {
                    $("#theone").live( "click", callback).html("现在点击它下方会出现文字");
                });
                $("#unbind").click(function () {
                    $("#theone").die( "click", callback).html("现在点击它一点反应也没有");
                    $(".hide_div").hide(1000);
                });
            });
