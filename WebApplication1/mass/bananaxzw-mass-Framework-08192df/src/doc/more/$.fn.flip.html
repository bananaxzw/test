$.fn.flip

描述：

一种翻滚动画。

参数：

duration
必需。Number或Object。时长或配置对象。
hash
可选。Object。配置对象。

返回值：

mass实例

hash中有如下可选参数:

键名值
beginBgc开始时的背景颜色。
endBgc结束时的背景颜色。
direction翻滚方向。有如下值:tb,bt,lr,rl.
before动画开始时的回调函数。
frame每次执行补间后执行的回调函数。
after动画结束时的回调函数。
例子
Hello! I'm a flip-box! :)
left top bottom right revert! 
        <div id="flipbox">Hello! I'm a flip-box! :)</div>
        <div id="flipPad">

            $.require("ready,fx,event,attr,more/flip", function(){
                $("#flipPad a:not(.revert)").click(function(){
                    var btn = $(this);
                    $("#flipbox").flip(1000,{
                        direction: btn.attr("rel"),
                        endBgc: btn.attr("rev"),
                        hyaline:"fff",
                        after: function(){
                            $(this).html( btn.attr("title") )
                        }
                    })
                    return false;
                });
            })
